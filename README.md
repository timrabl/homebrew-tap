# Homebrew Tap

Homebrew formulae for [timrabl](https://github.com/timrabl) projects.

## Install

```bash
brew tap timrabl/tap
brew install bbmctl
```

## Available formulae

| Formula | Description |
|---------|-------------|
| [bbmctl](https://github.com/timrabl/bbmctl) | CLI for the German Breitbandmessung broadband measurement API |
